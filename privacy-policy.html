Política de Privacidade — FIPE Car

Última atualização: 31 de maio de 2026

O aplicativo FIPE Car ("nós", "app") permite consultar preços da Tabela FIPE de veículos. Esta política descreve como tratamos informações quando você utiliza o app.

1. Dados que coletamos

  • Dados locais: nome de exibição, histórico de consultas, favoritos e comparações — armazenados apenas no seu dispositivo.
  • Dados de rede: consultas enviadas à API FIPE (marca, modelo, ano) para obter preços.
  • Publicidade: o app exibe anúncios via Google AdMob, que pode coletar identificadores de dispositivo e dados de uso conforme a política do Google.

2. Como usamos os dados

  • Exibir preços e histórico de veículos consultados.
  • Salvar favoritos e consultas recentes no dispositivo.
  • Exibir anúncios personalizados ou não personalizados (AdMob).
  • Melhorar a experiência do usuário.

3. Compartilhamento com terceiros

Podemos compartilhar dados com:

  • API FIPE (Parallelum): para consulta de preços de veículos.
  • Google AdMob: para exibição de anúncios. Política do Google
  • Wikimedia Commons: para busca de imagens de veículos.

Não vendemos dados pessoais.

4. Armazenamento e segurança

Dados de consultas e favoritos ficam no dispositivo via armazenamento local. Não mantemos servidor próprio com dados dos usuários.

5. Seus direitos

Você pode limpar histórico e favoritos nas configurações do app ou desinstalar o aplicativo para remover todos os dados locais.

6. Crianças

O app não é direcionado a menores de 13 anos.

7. Alterações

Esta política pode ser atualizada. A data no topo indica a versão vigente.
